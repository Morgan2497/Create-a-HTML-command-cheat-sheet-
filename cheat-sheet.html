HTML Tags Cheat Sheet

Headings

Headings from 'h1' to 'h6' identify blocks and secxtions. Every page must have one 'h1'.

Paragraphs

Paragraphs of text go in 'p' tags.

Lists

Unordered Lists

Unordered Lists are marked with bullet points and go in 'ul' tags.

Types of lists:

  • Unordered lists ('ul')
  • Ordered lists ('ol')
    •

    Ordered Lists

    Ordered lists are numbered and go in 'ol' tags. Each list item goes in an 'li' tag.

    Polya's Problem Solving Framework

    1. Understand the problem
    2. Come up with a plan
    3. Carry out the plan
    4. Go back and improve your solution

    Anchors (i.e. links)

    Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go the DuckDuckGo to search for more information.

    Setting 'target="_blank" will open the link in a new window. See MDN for more details.
